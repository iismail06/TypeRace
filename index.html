How to Use the Typing Test
  1. Select a difficulty (Easy, Medium, Hard) from the dropdown.
  2. Click Start. The passage fades out and the typing area activates.
  3. Type the passage exactly. Characters you type:
    • Green = correct character
    • Red = incorrect character
    • Grey (faint) text shows what is left to type.
  4. Click Stop when you finish (or stop early) to see:
    • Time (seconds)
    • WPM (whole number, only fully correct words counted)
    • Level (your chosen difficulty)
  5. Click Retry for a new random passage at the same difficulty, or change difficulty and Start again.

Tips:

  • Focus on accuracy first—speed follows.
  • Don’t backspace every tiny error; note patterns and improve next run.
  • Use the same posture & position for consistency.
